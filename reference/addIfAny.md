# Helper function to conditionally add elements to a list

Helper function to conditionally add elements to a list

## Usage

``` r
addIfAny(validation, name, value)
```

## Arguments

- validation:

  A list to which elements may be added.

- name:

  A character string representing the name of the element to add.

- value:

  The value to add to the list if it is not NULL or empty

## Value

The updated list with the new element added if applicable.

# Repair Missing IDs

This function repairs missing IDs in a pedigree.

## Usage

``` r
repairIDs(ped, verbose = FALSE)
```

## Arguments

- ped:

  A dataframe representing the pedigree data with columns \`ID\`,
  \`dadID\`, and \`momID\`.

- verbose:

  A logical flag indicating whether to print progress and validation
  messages to the console.

## Value

A corrected pedigree
